Dashboard Details How To
Connection using CGminer:

cgminer.exe -o [IP]:[PORT] -u "[Address]+[Diffculty]" -p [Pass] [CGminer settings]

[IP] :
[PORT] :
[Address] : Your wallet address
[Pass] : Anything
[CGminer settings] : your personal cgminer settings (for more info click: here)
[Difficulty] : Optional but recommended in order to get better stats.
(your hashrate in KHash/s)*0.00000116414

Example for a 1000 Khash/s:

cgminer.exe -o : -u 175tWpb8K1S7NmH4Zx6rewF9WQrcZv245W+0.00116414 -I 13
